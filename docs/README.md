# Documentation

This directory contains all documentation for the Scaffolding the Future project.

## Documentation Files

### Security Analysis
- **attack_vectors_analysis.md** - Comprehensive analysis of macOS attack vectors and how the sandbox solution addresses them
- **sandbox_flaws_analysis.md** - Critical security flaws identified in the sandbox implementation
- **sandbox_solutions.md** - Detailed solutions and recommendations for addressing identified flaws

### Guides
- **HARDENING_README.md** - macOS hardening guide with practical scripts
- **QUICK_START.md** - Quick start guide for getting started with the sandbox

### AI Integration
- **AGENT_FLOW.md** - AI agent workflow documentation
- **AGENT_PROMPTS.md** - AI agent prompts and interaction patterns

## Reading Order

1. Start with **QUICK_START.md** for installation and basic usage
2. Read **attack_vectors_analysis.md** to understand the threat model
3. Review **sandbox_flaws_analysis.md** to understand limitations
4. Check **sandbox_solutions.md** for improvement recommendations
5. Use **HARDENING_README.md** for host-level security hardening
